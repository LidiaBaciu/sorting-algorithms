# sorting-algorithms
C code for sorting algorithms

    What this program does:
    
    
    This program reads a number (let's name it 'n'), then a vector of n - elements.
    These elements are to be sorted by one of the following methods:
    1.1 Bubble sort
    1.2 Selection sort
    1.3 Insertion sort
    1.4 Merge sort
    1.5 Quick sort
